# Description

Simple browser extention that helps download videos from rezka.ag and some of its mirrors

### Installation

1. Download extension from (https://github.com/dkazankov/yard/releases/latest) and unpack.
2. Go to "Extentions" - "Manage extensions" and turn "Developer mode" on.
3. Click "Load unpacked" and select unpacked folder from 1.
   
### Usage

Extension adds green button to "Social media links" panel at the bottom of player.
Click on it and select link to download.
